Git

  • Use terminal to interact with GitHub using git syntax
  • Here are some common and essential commands to know:
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git add -A to stage any changes you make to your code
    • git commit -m "precise description of your changes" to commit those changes after being staged
    • git pull origin main to ensure that all stages and commits have been executed properly
    • git "push origin name of branch you are working on" (without the quotes) to push those changes onto your branch
    • navigate to GitHub on your browser to finish and merge code
