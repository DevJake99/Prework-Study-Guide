HTML

  • The head element contains information about the web page
  • The body element is where developers input the visible content that will be visible to the user
  • The Section element will create different sections of the webpage that we can later implement with CSS and Java

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Using selectors in CSS:
    • Selector { Property : Value;Property : Value; }
    • * is a selector in CSS used to apply the declarations inside the declaration box to all elements on that page

Git

  • Use terminal to interact with GitHub using git syntax
  • Here are some common and essential commands to know:
    • git status: checks what branch we are currently on
    • git checkout -b branch-name: creates a new branch and switches to it
    • git add -A to stage any changes you make to your code
    • git commit -m "precise description of your changes" to commit those changes after being staged
    • git pull origin main to ensure that all stages and commits have been executed properly
    • git "push origin name of branch you are working on" (without the quotes) to push those changes onto your branch
    • navigate to GitHub on your browser to finish and merge code
